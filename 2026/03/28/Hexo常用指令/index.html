二、hexo d 和 git push 是什么关系？

很多新手都会混淆这两个命令，其实它们完全不一样：

hexo d = 部署上线网站
作用：把生成的网页（public 文件夹）推送到 GitHub Pages
结果：别人能访问你的博客
属于：网站上线操作

git add + git commit + git push = 备份源码
作用：把你的文章、主题、配置文件备份到 GitHub 云端
结果：电脑坏了也能找回源码
属于：源码备份操作
