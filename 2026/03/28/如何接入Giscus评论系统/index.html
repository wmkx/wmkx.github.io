一、GitHub 仓库准备

  1. 进入 GitHub 仓库 → Settings → General → Features，勾选 Discussions 并保存。
  2. 在仓库顶部导航栏点击 Discussions 标签。
  3. 在页面左侧点击 Edit categories，选择现有分类（如 General）或新建 Comments 分类并保存。

二、安装并授权 Giscus App

安装地址：https://github.com/apps/giscus

  1. 点击 Install 安装。
  2. 选择 Only select repositories，指定用于评论的仓库。
  3. 完成授权安装。

三、生成 Giscus 配置代码

配置地址：https://giscus.app/zh-CN

  1. 输入仓库：用户名/仓库名（如：wmkx/wmkx.github.io）输入正确会提示“成功！该仓库满足所有条件。”
  2. 选择对应的 Discussion 分类
  3. 特性：默认勾选了第一项，可以增加第三、四项，看个人习惯，不重要。其余默认即可。
  4. 页面自动生成 script 代码
  5. 完整复制生成的配置脚本

四、集成到博客

打开博客根目录主题配置文件 _config.shiro.yml （此处主题名称为shiro），找到 giscus 配置：

giscus:
enable: true # 必须从 false 改为 true
repo: yourname/your-repo
repo-id: R_kgxxxxxx
category: Comments
category-id: DIC_kwxxxxxx
mapping: pathname
theme: light
lang: zh-CN
