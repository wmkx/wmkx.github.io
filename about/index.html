一、核心架构

  • 源站：GitHub Pages (wmkx/wmkx.github.io)
  • CDN & DNS：Cloudflare 免费版
  • 自定义域名：qingfengxiao.com（www.qingfengxiao.com）
  • 访问协议：HTTPS 强制 + HTTP/3 优先

二、✅ 已完成的关键配置

1. DNS 解析（Cloudflare）

配置项状态说明
www 子域名 CNAME已配置指向 wmkx.github.io
NS 服务器已切换Cloudflare 官方 NS
代理状态已开启（橙色云）隐藏源站 IP + CDN 加速

2. SSL/TLS 安全（Cloudflare）

配置项状态说明
加密模式Full (strict)最高安全级别
Always Use HTTPS已开启强制 HTTP 跳转 HTTPS
Automatic HTTPS Rewrites已开启自动修复混合内容

3. 性能与可用性优化（Cloudflare）

配置项状态说明

HTTP/2已开启兼容旧设备
Always Online已开启故障时展示缓存页面
IPv6 Compatibility已开启支持 IPv6 访问

三、🎯 核心优势总结

  1. 抗 DNS 污染：Cloudflare 接管 DNS，隐藏源站 IP，避免国内 DNS 污染
  2. 全球加速：Cloudflare Anycast 网络，让全球访客都能快速访问
  3. 高可用性：Always Online + GitHub Pages 冗余，保证网站极少下线
  4. 安全可靠：全站 HTTPS + HSTS，防止劫持与内容篡改
  5. 免费高效：所有服务均为免费版，足够个人博客长期使用
